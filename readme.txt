Парсер для my-hit.org работает достаточно быстро. После запуска отображается нумерация и список жанров, подсчитывается сколько страниц с фильмами есть у каждого жанра. Далее следует ввести номер жанра фильма которые следует парсить. После спрашивается сколько страниц следует спарсить. Так же стоит "блокировка" на ввод числа который не соответствует жанру, и ввод числа который превышает количество страниц.  За 2 минуты данный парсер вытащил 9000 фильмов 1 жанра. В корневой директории создается файлик parser.xml который следует открыть с помощью excel.
